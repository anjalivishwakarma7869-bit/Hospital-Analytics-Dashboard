# 🏥 Hospital Analytics Dashboard

## 📌 Project Overview

This project is an interactive Hospital Analytics Dashboard built using Power BI. The dashboard provides insights into patient management, appointment tracking, hospital revenue, treatment costs, and payment status. It helps hospital administrators monitor performance and make data-driven decisions.

---

## 🎯 Business Problem

Hospitals generate large amounts of operational data every day. Managing appointments, tracking revenue, monitoring treatment costs, and analyzing patient demographics can be challenging without proper visualization.

This dashboard helps answer key business questions such as:

- How many patients visited the hospital?
- How many appointments were completed?
- What is the total revenue generated?
- Which treatments have the highest costs?
- What is the payment status of patients?
- What is the gender distribution of patients?

---

## 📈 Dashboard Visuals

### 1. Doctor-wise Patient Handling
Analyzes the number of patients handled by each doctor.

### 2. Appointment Status Distribution
Shows appointment status breakdown:
- Completed
- Scheduled
- Cancelled
- No-show

### 3. Monthly Revenue Trend
Tracks hospital revenue across different months.

### 4. Treatment Cost Analysis
Compares treatment costs across different treatment types.

### 5. Patient Gender Distribution
Displays male and female patient proportions.

### 6. Interactive Filters
Users can filter data using:
- Payment Method
- Doctor ID
- Treatment Type
